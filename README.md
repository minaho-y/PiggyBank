# 🐷 PiggyBank

PiggyBankは，目標金額に向けた貯金の進捗を管理できるSwiftUIアプリです．
目標を設定し，入金を記録することで，現在の貯金額や達成率を確認できます．

## 主な機能

- 貯金目標の設定
- 入金記録の追加・編集・削除
- 現在の貯金額，達成率，残り金額の表示
- 円グラフによる達成率の可視化
- 入金履歴の一覧表示

## 使用技術

- Swift
- SwiftUI
- SwiftData
- Charts

## 起動方法

1. `PiggyBank.xcodeproj` をXcodeで開く
2. シミュレータまたは実機を選択
3. `Command + R` で実行
